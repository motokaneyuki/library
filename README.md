# Purpose
This project was created to practice using object constructors.

# Challenges faced
It was definitely a challenge to figure out how to isolate state changes so that updating or deleting one specific book card didn't break the rest of the cards. I learned a lot about syncing the UI with my data array by using unique IDs for the button click handlers.
